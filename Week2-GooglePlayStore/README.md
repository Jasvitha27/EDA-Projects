# Google Play Store - Exploratory Data Analysis

## 📌 Objective
To analyze app ratings, categories, reviews, installs, and pricing trends in the Google Play Store dataset.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🔎 Analysis Performed
- Data cleaning and preprocessing
- Handling missing values
- Rating distribution analysis
- Category-wise comparison
- Install and review trends

## 📊 Key Insights
- Free apps dominate the Play Store.
- Certain categories have consistently higher ratings.
- Reviews and installs show strong correlation.

## 📁 Files Included
- googleplaystore.ipynb → Complete EDA notebook
- googleplaystore.csv → Dataset used
