
# Students Performance - Exploratory Data Analysis

## 📌 Objective
To analyze factors affecting students' academic performance using data visualization and statistical insights.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🔎 Analysis Performed
- Data cleaning
- Checking missing values
- Distribution of scores
- Correlation analysis
- Gender-wise performance comparison

## 📊 Key Insights
- Math, reading, and writing scores are positively correlated.
- Gender differences observed in subject performance.
- Parental education level impacts student scores.

## 📁 Files Included
- Students_performance_csv.ipynb → Complete EDA notebook
- StudentsPerformance.xls → Dataset used
