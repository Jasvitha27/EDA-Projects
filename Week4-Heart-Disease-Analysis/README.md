
# Heart Disease - Exploratory Data Analysis

## 📌 Objective
To analyze medical attributes and identify patterns associated with heart disease risk.

## 📂 Dataset Overview
The dataset contains patient health metrics such as:
- Age
- Sex
- Chest pain type
- Cholesterol
- Blood pressure
- Maximum heart rate
- Target (heart disease presence)

## 🧹 Data Cleaning
- Checked missing values
- Verified data types
- Removed duplicates (if any)

## 📊 Analysis Performed
- Distribution of age and cholesterol
- Relationship between chest pain and target
- Correlation heatmap
- Gender-based comparison
- Risk factor analysis

## 📈 Key Insights
- Higher cholesterol is associated with higher risk
- Certain chest pain types strongly indicate heart disease
- Age plays a significant role

## 🏁 Conclusion
The analysis highlights important medical indicators that contribute to heart disease risk.
